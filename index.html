Bab 1: HTML

1. Struktur Dasar HTML

Struktur dasar HTML selalu dimulai dengan <!DOCTYPE html> yang memberitahu browser bahwa ini adalah dokumen HTML5.
Kemudian ada <html> sebagai wadah utama,
<head> untuk informasi meta (tidak terlihat di halaman), dan
<body> untuk konten yang terlihat.

<!DOCTYPE html> <html> <head> <title>Halaman Pertamaku</title> </head> <body> <h1>Selamat Datang!</h1> <p>Ini adalah paragraf pertama.</p> </body> </html>
Output di Browser:
[ Judul Tab: Halaman Pertamaku ] Selamat Datang! Ini adalah paragraf pertama.

2. Format Teks dan Heading

Heading adalah judul atau subjudul dalam halaman web. HTML menyediakan 6 level heading dari h1 (paling besar dan penting) sampai h6 (paling kecil). Gunakan h1 untuk judul utama halaman, h2 untuk subjudul, dan seterusnya sesuai hierarki.

Tag <strong> membuat teks tebal (bold), sementara <em> membuat teks miring (italic).

<h1>Heading Level 1 - Judul Utama</h1> <h2>Heading Level 2 - Subjudul</h2> <h3>Heading Level 3 - Sub-subjudul</h3> <p>Paragraf biasa dengan teks.</p> <p>Ini teks <strong>tebal</strong> dan ini <em>miring</em>.</p>
Output di Browser:
Heading Level 1 - Judul Utama Heading Level 2 - Subjudul Heading Level 3 - Sub-subjudul Paragraf biasa dengan teks. Ini teks tebal dan ini miring.

3. Hyperlink - Membuat Tautan

Hyperlink atau link adalah elemen yang membuat web menjadi "web" - menghubungkan satu halaman ke halaman lain.
Tag <a> (anchor) digunakan dengan atribut href (hypertext reference) yang menentukan tujuan link.

Atribut target="_blank" membuat link terbuka di tab baru. Untuk link ke bagian dalam halaman yang sama, gunakan href="#namaID" yang akan melompat ke elemen dengan id tersebut.

<!-- Link ke website eksternal --> <a href="https://google.com">Pergi ke Google</a> <!-- Link ke halaman lokal --> <a href="about.html">Tentang Kami</a> <!-- Link ke bagian halaman (anchor) --> <a href="#section1">Ke Bagian 1</a> <!-- Link yang membuka di tab baru --> <a href="https://github.com" target="_blank">GitHub</a>
Output di Browser:
Pergi ke Google Tentang Kami Ke Bagian 1 GitHub (terbuka di tab baru)

4. Menambahkan Gambar

Tag <img> digunakan untuk menampilkan gambar di halaman web. Atribut src (source) menentukan lokasi file gambar - bisa dari komputer kita atau dari internet (URL).

Atribut alt (alternative text) adalah teks yang muncul jika gambar gagal dimuat.
Atribut width dan height mengatur ukuran gambar dalam pixel.

<!-- Gambar dari file lokal --> <img src="foto.jpg" alt="Foto pemandangan" width="300"> <!-- Gambar dari URL internet --> <img src="https://example.com/image.png" alt="Gambar contoh" width="400" height="300"> <!-- Gambar yang bisa diklik (dengan link) --> <a href="https://example.com"> <img src="logo.png" alt="Logo perusahaan"> </a>
Output di Browser:
[Gambar pemandangan ditampilkan dengan lebar 300px] [Gambar dari internet ditampilkan 400x300px] [Logo yang bisa diklik untuk ke example.com]

5. List - Membuat Daftar

List atau daftar digunakan untuk mengelompokkan item-item yang berhubungan. HTML menyediakan 3 jenis list dengan fungsi berbeda:

â€¢ Unordered List (<ul>) - Daftar tanpa urutan spesifik, ditandai dengan bullet point (â€¢).

â€¢ Ordered List (<ol>) - Daftar berurutan dengan nomor otomatis (1, 2, 3...).

â€¢ Description List (<dl>) - Daftar istilah dan penjelasannya, seperti kamus mini. Menggunakan <dt> (definition term) untuk istilah dan <dd> (definition description) untuk penjelasannya.

â€¢ List Item (<li>) - Ini adalah item individual dalam ul atau ol. Setiap baris/poin dalam daftar harus dibungkus dengan tag <li>.

6. Tabel - Menampilkan Data Terstruktur

Mirip kayak spreadsheet Excel. Tabel terdiri dari beberapa elemen:

â€¢ <table> - Wadah utama untuk seluruh tabel

â€¢ <tr> (table row) - Mendefinisikan satu baris horizontal dalam tabel

â€¢ <th> (table header) - Sel header/judul kolom, biasanya ditampilkan tebal dan di tengah

â€¢ <td> (table data) - Sel berisi data/isi tabel

8. Pengenalan CSS - Styling Website

CSS (Cascading Style Sheets) adalah bahasa untuk mempercantik tampilan website. Jika HTML adalah kerangka/tulang, maka CSS adalah kulit dan pakaiannya. CSS mengatur hampir semua aspek visual website.

Ada 3 cara menulis CSS: Inline CSS (langsung di tag HTML dengan atribut style), Internal CSS (di dalam tag <style> di bagian <head>), dan External CSS (file .css terpisah). Internal CSS yang paling sering digunakan untuk belajar.

CSS bekerja dengan selector (memilih elemen mana yang mau di-style) dan property (apa yang mau diubah). Format dasarnya: selector { property: value; }

Bab 2: C++

1. Struktur Dasar C++

Setiap program C++ harus punya fungsi main() sebagai titik mulai eksekusi program.

#include <iostream> berfungsi untuk memasukkan library input-output standar.
using namespace std; memungkinkan kita menggunakan cout dan cin tanpa menulis std:: di depannya.
return 0; memberitahu sistem operasi bahwa program berjalan sukses tanpa error.

#include <iostream> using namespace std; int main() { cout << "Hello, World!" << endl; cout << "Selamat datang di C++!" << endl; return 0; }
Output Console:
Hello, World! Selamat datang di C++!

2. Input dan Output

Input dan output adalah cara program berkomunikasi dengan user. cout (console output) digunakan untuk menampilkan data ke layar, sementara cin (console input) digunakan untuk membaca input dari keyboard.

Operator << (insertion operator) "memasukkan" data ke cout untuk ditampilkan. Operator >> (extraction operator) "mengekstrak" data dari cin ke variabel. endl adalah end line yang membuat baris baru (seperti menekan Enter).

// Output - Menampilkan pesan cout << "Masukkan nama Anda: "; // Input - Membaca input nama cin >> nama; cout << "Masukkan umur Anda: "; cin >> umur; // Output hasil dengan menggabungkan teks dan variabel cout << "Halo " << nama << ", umur kamu " << umur << " tahun" << endl; cout << "Selamat belajar C++!" << endl;
Output Console:
Masukkan nama Anda: Budi Masukkan umur Anda: 17 Halo Budi, umur kamu 17 tahun Selamat belajar C++!

3. Tipe Data dan Variabel

Variabel adalah wadah untuk menyimpan data. Setiap variabel punya tipe data yang menentukan jenis nilai apa yang bisa disimpan dan berapa banyak memori yang digunakan.

int untuk bilangan bulat (tidak ada koma),
float untuk bilangan desimal presisi tunggal,
double untuk desimal presisi ganda (lebih akurat),
char untuk satu karakter (harus pakai petik tunggal),
string untuk teks/kumpulan karakter, dan
bool untuk nilai benar/salah (true/false).

#include <iostream> using namespace std; int main() { // Deklarasi dan inisialisasi variabel int angka = 10; // Bilangan bulat float desimal = 3.14; // Bilangan desimal double presisi = 3.14159265; // Desimal presisi tinggi char huruf = 'A'; // Karakter tunggal string teks = "Hello"; // Kumpulan karakter bool benar = true; // Boolean true atau false // Menampilkan semua variabel cout << "Integer: " << angka << endl; cout << "Float: " << desimal << endl; cout << "Double: " << presisi << endl; cout << "Char: " << huruf << endl; cout << "String: " << teks << endl; cout << "Bool: " << benar << endl; return 0; }
Output Console:
Integer: 10 Float: 3.14 Double: 3.14159265 Char: A String: Hello Bool: 1

4. Percabangan IF-ELSE

Percabangan adalah kemampuan program untuk membuat keputusan. IF-ELSE memungkinkan program menjalankan kode berbeda berdasarkan kondisi tertentu - seperti "jika nilai lebih dari 80, tampilkan A, jika tidak tampilkan B".

Struktur: if (kondisi) dijalankan jika kondisi benar, else if (kondisi2) untuk kondisi alternatif, dan else dijalankan jika semua kondisi di atasnya salah.

Operator perbandingan:
>= (lebih besar sama dengan),
<= (lebih kecil sama dengan),
== (sama dengan),
!= (tidak sama dengan).

int main() { int nilai; cout << "Masukkan nilai ujian (0-100): "; cin >> nilai; // Struktur IF-ELSE untuk menentukan grade if (nilai >= 80) { cout << "Grade: A - Sangat Baik!" << endl; cout << "Pertahankan prestasimu!" << endl; } else if (nilai >= 70) { cout << "Grade: B - Baik" << endl; cout << "Tingkatkan lagi ya!" << endl; } else if (nilai >= 60) { cout << "Grade: C - Cukup" << endl; cout << "Masih bisa lebih baik!" << endl; } else { cout << "Grade: D - Kurang" << endl; cout << "Harus lebih giat belajar!" << endl; } return 0; }
Output Console (jika input 85):
Masukkan nilai ujian (0-100): 85 Grade: A - Sangat Baik! Pertahankan prestasimu!
Output Console (jika input 65):
Masukkan nilai ujian (0-100): 65 Grade: C - Cukup Masih bisa lebih baik!

5. Percabangan SWITCH-CASE

SWITCH-CASE adalah alternatif IF-ELSE yang lebih rapi ketika kita punya banyak pilihan berdasarkan satu variabel. Bayangkan seperti menu restoran - pilih nomor 1, 2, atau 3 dan dapat makanan berbeda.

switch (variabel) mengecek nilai variabel, case nilai: adalah pilihan yang mungkin, break; menghentikan eksekusi (keluar dari switch) setelah case dijalankan, dan default: dijalankan jika tidak ada case yang cocok (seperti else di IF-ELSE).

int main() { int pilihan; int a = 10, b = 5; cout << "=== KALKULATOR SEDERHANA ===" << endl; cout << "1. Penjumlahan" << endl; cout << "2. Pengurangan" << endl; cout << "3. Perkalian" << endl; cout << "4. Pembagian" << endl; cout << "Pilih operasi (1-4): "; cin >> pilihan; switch (pilihan) { case 1: cout << "Hasil: " << a << " + " << b << " = " << (a + b) << endl; break; case 2: cout << "Hasil: " << a << " - " << b << " = " << (a - b) << endl; break; case 3: cout << "Hasil: " << a << " * " << b << " = " << (a * b) << endl; break; case 4: cout << "Hasil: " << a << " / " << b << " = " << (a / b) << endl; break; default: cout << "Pilihan tidak valid! Masukkan 1-4" << endl; } return 0; }
Output Console (jika pilih 3):
=== KALKULATOR SEDERHANA === 1. Penjumlahan 2. Pengurangan 3. Perkalian 4. Pembagian Pilih operasi (1-4): 3 Hasil: 10 * 5 = 50

6. Perulangan FOR

Perulangan FOR adalah cara menjalankan blok kode berulang kali sejumlah tertentu. Sangat berguna ketika kita tahu pasti berapa kali harus mengulang - misalnya mencetak angka 1-10 atau membuat pola bintang.

Struktur: for (inisialisasi; kondisi; increment).
Inisialisasi menentukan nilai awal (biasanya i=0 atau i=1),
kondisi menentukan kapan berhenti (i<=10), dan
increment menentukan bagaimana variabel berubah setiap iterasi (i++ artinya tambah 1).

Nested loop (perulangan bersarang) adalah loop di dalam loop - seperti membuat pola bintang, loop luar untuk baris dan loop dalam untuk kolom.

int main() { // Perulangan sederhana - mencetak angka cout << "Angka 1 sampai 5:" << endl; for (int i = 1; i <= 5; i++) { cout << "Perulangan ke-" << i << endl; } // Nested loop - membuat pola bintang segitiga cout << "\nPola Bintang Segitiga:" << endl; for (int i = 1; i <= 5; i++) { for (int j = 1; j <= i; j++) { cout << "* "; } cout << endl; } return 0; }
Output Console:
Angka 1 sampai 5: Perulangan ke-1 Perulangan ke-2 Perulangan ke-3 Perulangan ke-4 Perulangan ke-5 Pola Bintang Segitiga: * * * * * * * * * * * * * * *

6.5. Nested Loop (Perulangan Bersarang)

Nested loop adalah perulangan di dalam perulangan. Loop luar mengontrol baris, loop dalam mengontrol kolom. Konsep ini sering dipakai untuk membuat pola, matriks, atau tabel.

#include <iostream> using namespace std; int main() { // POLA 1: Segitiga Angka cout << "Pola Segitiga Angka:" << endl; for (int i = 1; i <= 4; i++) { // Loop luar: baris 1-4 for (int j = 1; j <= i; j++) { // Loop dalam: kolom 1 sampai i cout << j << " "; } cout << endl; // Pindah baris setelah loop dalam selesai } // POLA 2: Persegi Panjang Bintang cout << "\nPola Persegi Panjang:" << endl; for (int i = 1; i <= 3; i++) { // 3 baris for (int j = 1; j <= 5; j++) { // 5 kolom cout << "* "; } cout << endl; } // POLA 3: Tabel Perkalian cout << "\nTabel Perkalian 2x2:" << endl; for (int i = 1; i <= 2; i++) { for (int j = 1; j <= 2; j++) { cout << i << " x " << j << " = " << (i*j) << "\t"; } cout << endl; } // POLA 4: Segitiga Terbalik cout << "\nSegitiga Terbalik:" << endl; for (int i = 4; i >= 1; i--) { // Mulai dari 4 turun ke 1 for (int j = 1; j <= i; j++) { cout << "* "; } cout << endl; } return 0; }

7. Perulangan WHILE

Perulangan WHILE digunakan ketika kita tahu kondisi berhentinya tapi tidak tahu pasti berapa kali akan mengulang. While akan terus mengulang SELAMA kondisinya masih true (benar).

Bedanya dengan FOR: FOR untuk jumlah iterasi yang sudah pasti, WHILE untuk iterasi yang bergantung kondisi. Contoh: terus minta input sampai user ketik 'n', atau terus hitung sampai angka mencapai nilai tertentu.

PENTING: Pastikan kondisi bisa berubah menjadi false, jika tidak program akan loop selamanya (infinite loop)!

int main() { int angka = 1; // While loop sederhana cout << "Hitungan mundur dari 5:" << endl; int hitung = 5; while (hitung >= 1) { cout << hitung << "... "; hitung--; } cout << "Selesai!" << endl << endl; // While dengan input user char jawab = 'y'; int counter = 0; while (jawab == 'y' || jawab == 'Y') { counter++; cout << "Loop ke-" << counter << ": Halo!" << endl; cout << "Ulangi lagi? (y/n): "; cin >> jawab; } cout << "Program berhenti. Total loop: " << counter << endl; return 0; }
Output Console:

8. Perulangan DO-WHILE

DO-WHILE mirip dengan WHILE, tetapi dengan satu perbedaan krusial: DO-WHILE mengecek kondisi SETELAH menjalankan kode, sehingga dijamin dijalankan minimal 1 kali. While mengecek kondisi SEBELUM menjalankan kode.

Kapan pakai DO-WHILE? Ketika kita ingin kode dijalankan dulu minimal sekali baru cek kondisi. Contoh: menu program yang harus tampil dulu sebelum user memilih keluar, atau validasi input yang harus dicoba minimal sekali.

Struktur: do { kode } while (kondisi); - perhatikan titik koma setelah while!

int main() { int angka; int jumlah = 0; int count = 0; cout << "=== PROGRAM PENJUMLAHAN ===" << endl; cout << "Masukkan angka untuk dijumlahkan" << endl; cout << "(Masukkan 0 untuk selesai dan lihat hasil)" << endl << endl; // Do-while akan jalan minimal 1 kali do { cout << "Masukkan angka: "; cin >> angka; if (angka != 0) { jumlah += angka; // Tambahkan ke total count++; cout << "Total sementara: " << jumlah << endl; } } while (angka != 0); // Terus ulang sampai user input 0 cout << "\n=== HASIL ===" << endl; cout << "Total angka yang dimasukkan: " << count << endl; cout << "Jumlah total: " << jumlah << endl; if (count > 0) { cout << "Rata-rata: " << (float)jumlah/count << endl; } cout << "Program selesai!" << endl; return 0; }
Output Console:
=== PROGRAM PENJUMLAHAN === Masukkan angka untuk dijumlahkan (Masukkan 0 untuk selesai dan lihat hasil) Masukkan angka: 10 Total sementara: 10 Masukkan angka: 25 Total sementara: 35 Masukkan angka: 15 Total sementara: 50 Masukkan angka: 0 === HASIL === Total angka yang dimasukkan: 3 Jumlah total: 50 Rata-rata: 16.6667 Program selesai!
